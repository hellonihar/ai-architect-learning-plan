# New AI Project

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\Activate on Windows
pip install -r requirements.txt
```

## Structure

```
src/
  __init__.py
  main.py
tests/
  test_main.py
```

## Running

```bash
python src/main.py
```

## Testing

```bash
pytest tests/
```
